# Scene Modules (`Prefabs/SceneModules/`)

Reusable world-space gameplay modules (not level-specific scripts).

| Subfolder | Examples |
|-----------|----------|
| `Interactables/` | ReturnButton, pressure plates |
| `BuffStations/` | Heal / Stamina / Shield buff bubbles |
| `Portals/` | Level exit, hub advance portal |
| `Stations/` | Save points, terminals |
| `ResourceNodes/` | Wood, stone, ore nodes |
| `Buildables/` | Workbench, furnace, chest |
| `Gates/` | Locked gates, energy barriers |
| `Tutorial/` | Signs, direction boards |

Shared labels/VFX: `Prefabs/Shared/`

See `Documentation/SceneModules/Scene_Module_Migration_Plan.md`.
