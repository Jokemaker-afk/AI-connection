# Prefabs (`Assets/_Project/Prefabs/`)

Runtime-ready prefabs. Level 8 modular environment prefabs live under `Environment/`, `Props/`, and `Level8/`.

| Subfolder | Purpose |
|-----------|---------|
| `Environment/` | Chunk prefabs, terrain patches, walls, boundaries, connectors |
| `Props/` | Decorations, resources, hazards, objectives (by category) |
| `Level8/` | Level-8-specific assembled prefab aliases / bundles |
| `Shared/` | Labels, interaction markers, shared VFX |
| `Enemies/` | Enemy placeholders (existing) |

Naming: `PF_Biome_Category_Name_Variant` — see Environment Art documentation.
